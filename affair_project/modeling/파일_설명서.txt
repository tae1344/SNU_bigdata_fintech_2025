불륜 발생 가능성을 예측하는 모델을 개발했습니다.

📋 제출 요구사항별 파일 매핑:
===============================================================================
1️⃣ 사용한 dataset (전처리 후)
   📄 파일명: affairs.csv
   📊 내용: 
      - 총 6,367개의 레코드
      - 9개 특성 변수 (rate_marriage, age, yrs_married, children, 
        religious, educ, occupation, occupation_husb, affairs)
      - 전처리 완료된 깨끗한 데이터셋
      - 결측치 없음, 이상치 처리 완료

2️⃣ 코드(.ipynb)
   📄 파일명: 
      - EDA.ipynb (탐색적 데이터 분석)
      - mini_affairs.ipynb (기본 모델링)
      - mini_affairs_adj.ipynb (고급 모델링)
   
   🔍 각 노트북의 역할:
      * EDA.ipynb: 데이터 탐색, 시각화, 상관관계 분석
      * mini_affairs.ipynb: 로지스틱 회귀를 이용한 기본 분류 모델
      * mini_affairs_adj.ipynb: 피처 엔지니어링과 하이퍼파라미터 튜닝이 포함된 고급 모델

3️⃣ 결과 테이블 (csv)
   📄 파일명: 
      - modeling_results.csv (주요 성능 지표)
      - detailed_model_comparison.csv (모델별 상세 비교)
   
   📊 포함 내용:
      * Accuracy, Precision, Recall, F1-Score, ROC AUC
      * 최적 임계값과 선택된 모델 정보
      * 로지스틱 회귀 vs 랜덤 포레스트 성능 비교

4️⃣ 발표 슬라이드 (pdf)
   📄 파일명: 1조_발표자료.pdf
   📖 내용: 25페이지 분량의 프로젝트 발표 자료

===============================================================================
🔧 기술적 세부사항:
===============================================================================

📈 모델링 접근법:
   - 이진 분류 문제 (불륜 유무 예측)
   - 불균형 데이터 처리 (class_weight='balanced' 사용)
   - 5-fold 교차 검증
   - 하이퍼파라미터 튜닝 (GridSearchCV)
   - 임계값 최적화

🛠️ 사용된 기술:
   - Python, pandas, numpy, scikit-learn
   - 피처 엔지니어링 (yrs_per_age, rate_x_yrs)
   - 범주형 변수 인코딩 (OneHotEncoder)
   - 수치형 변수 정규화 (StandardScaler)
   - 파이프라인 구축

===============================================================================
📊 주요 결과:
===============================================================================

🏆 최적 모델: Random Forest
   - Accuracy: 65.46%
   - Precision: 47.46%
   - Recall: 65.94%
   - F1-Score: 55.19%
   - ROC AUC: 72.09%
   - 최적 임계값: 0.46

🔍 주요 발견사항:
   - 결혼 만족도가 낮을수록 불륜 가능성 증가
   - 종교성이 낮을수록 불륜 가능성 증가
   - 나이와 결혼 기간이 불륜과 음의 상관관계
